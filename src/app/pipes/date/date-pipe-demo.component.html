日期格式化pipe

示例

  • 原日期：2017/12/31 23:59:59
  • 格式化1：{{'2017/12/31 23:59:59'|date:'yyyy-MM-dd hh:mm:ss'}}
  • 格式化2：{{'2017/12/31 23:59:59'|date:'yyyy年MM月dd日 hh时mm分ss秒'}}
  • 格式化3：{{'2017/12/31 23:59:59'|date:'yyyy-MM-dd'}}
  • ...

参数

参数名 类型 说明
format string 格式。yyyy或YYYY代表年，MM代表月，dd或DD代表日，hh或HH代表时，mm代表分，ss或SS代表秒； 如：yyyy年MM月dd日 hh时mm分ss秒、yyyy-MM-dd hh:mm:ss
