地址选择directive

类名 AreaPickerDirective

如何使用

1.areaPicker指令和AreaPicker实例配合使用；

2.在input元素上使用areaPicker指令；

3.areaPicker指向一个AreaPicker实例。
所属模块 AreaPickerModule
类型 指令
selector areaPicker

示例

  • html
  • typescript

areaPicker指令属性

属性名 类型 输入/输出类型 说明
areaPicker Input AreaPicker AreaPicker实例。

AreaPicker属性

属性名 类型 说明
init Function 弹出选择框时触发的事件。在该方法内调用setData方法设置items第一个元素data值。
items { label?: string, elem?: HTMLElement, key: string, data: any[], selected: Function }[]

label:选项标签

key:选项名读取的data的key

data:选项数据

selected:当前选项卡选择完毕触发的事件，在该方法内设置下一选项的数据
done Function 选择完毕事件，传入参数为选择的各个data的元素
