{{checkbox1?'已选中':'未选中'}} {{checkbox2?'已选中':'未选中'}} {{checkbox3?'已选中':'未选中'}} {{checkbox4?'已选中':'未选中'}} {{checkbox5?'已选中':'未选中'}} {{checkbox6?'已选中':'未选中'}} {{checkbox7?'已选中':'未选中'}}

@Input

属性名 类型 说明
value boolean 输入值。默认false。
size string 组件尺寸。默认md,可选：lg、md、sm。
display string 显示方式。默认为行内，可选block（块级）
disabled any 绑定在组件内input元素上disabled属性的值。
innerStyle string 框内的样式，默认为正方形 ，可选 checked。
customBackground string 勾选时的背景，值为颜色值，如rgba(0,0,0,1)、#000000。
styleClass string 绑定在组件内最外层元素上的css类。
name string 绑定在组件内input元素上name属性的值。

@Output

属性名 类型 说明
valueChange 输入值改变事件，触发时传入参数为value的值。
