模态框 component

类名

ModalComponent

ModalHeaderComponent

ModalBodyComponent

ModalFooterComponent

关系：

1.modal是父组件，modal-header、modal-body、modal-footer是子组件；

2.子组件应被包含在父组件中。

两种使用方式：

1.通过属性visible双向绑定控制显示/隐藏；

2.通过声明模版变量（指向组件实例），调用open/close方法控制显示/隐藏。

modal属性

属性名 类型 输入/输出类型 说明
visible Input boolean 控制显示和隐藏
visibleChange Output EventEmitter < any> visible改变事件；触发时传入参数为visible。
onOpen Output EventEmitter < any> 模态框从隐藏到显示完成时的事件；触发时传入参数为visible。
onClose Output EventEmitter < any> 模态框关闭事件；触发时传入参数为visible。
overflow Input boolean 是否溢出屏幕，默认为false。
animated Input boolean 是否有过渡动画，默认为true。
size Input string 模态框尺寸，默认为md，可选lg、md、sm。
styleClass Input string 添加到组件内最外层元素上的css类。
fullHeight Input boolean 是否默认撑满屏幕高度；默认为false。
disabled Input boolean 是否禁用（有遮罩）；默认为false。
outClickClose Input boolean 是否点击空白处关闭；默认为false。
showCloseButton Input boolean 是否显示关闭按钮；默认为true。
width Input number 模态框宽度。
