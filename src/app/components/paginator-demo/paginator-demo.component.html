@Input

属性名 类型 说明
count number 总条目。
pageSize number 每页大小。
index number 当前页；初始值为0。
pageCount number 总页数。
maximum number 最多显示的按钮数，默认为5。
changePageSizeAble boolean 是否可改变每页大小，默认为false。
inputAble boolean 是否可输入页码进行跳转，默认为false。
pageSizeOptions number[] 每页大小的选项。
size string 组件尺寸；除默认尺寸外可选项：sm
showTotal boolean 是否显示文字信息；默认为false。
styleClass string 添加到组件内最外层元素的css类。

@Output

属性名 类型 说明
pageSizeChange EventEmitter < any> pageSize改变事件；触发时传入参数为pageSize。
indexChange EventEmitter < any> index改变事件；触发时传入参数为index。
onChangePage EventEmitter < any> 页面index改变后触发的事件，触发时传入参数为index。
onChangePageError EventEmitter < any> 页面跳转失败时触发的事件，触发时传入参数为输入的页码或index。
