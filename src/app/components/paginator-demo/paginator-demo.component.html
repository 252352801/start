分页组件 component

类名 PaginatorComponent
所属模块 PaginatorModule
类型 组件
selector paginator

示例

  • html
  • typescript

属性

属性名 类型 输入/输出类型 说明
count Input number 总条目。
pageSize Input number 每页大小。
pageSizeChange Output EventEmitter< any > pageSize改变事件；触发时传入参数为pageSize。
index Input number 当前页；初始值为0。
indexChange Output EventEmitter< any > index改变事件；触发时传入参数为index。
pageCount Input number 总页数。
maximum Input number 最多显示的按钮数，默认为5。
changePageSizeAble Input boolean 是否可改变每页大小，默认为false。
inputAble Input boolean 是否可输入页码进行跳转，默认为false。
pageSizeOptions Input number[] 每页大小的选项。
size Input string 组件尺寸；除默认尺寸外可选项：sm
showTotal Input boolean 是否显示文字信息；默认为false。
styleClass Input string 添加到组件内最外层元素的css类。
onChangePage Output EventEmitter< any > 页面index改变后触发的事件，触发时传入参数为index。
onChangePageError Output EventEmitter< any > 页面跳转失败时触发的事件，触发时传入参数为输入的页码或index。
